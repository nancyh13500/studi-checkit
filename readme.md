# CheckIt "To Do Lsit"

Application web de gestion de listes de tâches en ligne.

## Description

CheckIt est une plateforme permettant de créer et gérer des listes de tâches. Vous pouvez créer un nombre illimité de listes, les classer par catégories et les retrouver facilement.
